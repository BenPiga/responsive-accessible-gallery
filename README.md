#Responsive & Accessible Photo Gallery

## Overview
This project is a fully responsive, mobile-first photo gallery built with HTML5 and CSS3. It demonstrates the use of advanced CSS media queries to adapt the layout across different screen sizes (mobile, tablet, and desktop) while strictly adhering to web accessibility standards.

## Key Features
> Responsive CSS Grid: Transforms from a 1-column layout (mobile) to a 2-column layout (tablet), and finally a dynamic 3-column layout (desktop) using fluid viewport calculations.
> Dark Mode Support: Automatically detects the user's system preference (`prefers-color-scheme: dark`) and applies a custom high-contrast dark theme.
> Reduced Motion: Detects the user's system preference for reduced animations (`prefers-reduced-motion: reduce`) and disables the smooth-scrolling feature to accommodate users with motion sensitivities.
> Accessible & Validated: Uses semantic HTML, descriptive `alt` text for all images, and passes both the W3C Markup Validation and WAVE Accessibility tools.

## Technologies Used
> HTML5
> CSS3 (CSS Grid, Media Queries, System Preference Queries)
